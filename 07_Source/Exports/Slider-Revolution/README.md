# Slider Revolution Export

## What belongs here

- Exported **Slider Revolution modules** — the audit recorded **12 modules** in
  active use across the site.
- A short note recording the export date, plugin version, and method.

## What does not belong here

- The Slider Revolution plugin itself (vendor product).
- Media assets that already live in `02_Assets/` — reference, don't duplicate.

> **Caution:** the audit noted **no active licence key**, which may limit export.
> Verify a full module export is possible before relying on it, and capture
> before any migration. See the
> [Customisation Audit](../../../01_Documentation/CUSTOMISATION_AUDIT.md).
