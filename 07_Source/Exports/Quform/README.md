# Quform Export

## What belongs here

- Exported **Quform form definitions** (General Contact Form, Tailor-made
  Holidays Form).
- Exported **form entries** — the audit recorded **190 live enquiry/booking
  entries**.
- A short note recording the export date, Quform version, and method.

## What does not belong here

- Custom code or unrelated plugin data.
- Raw entry data left unprotected — this contains personal/customer information;
  store and handle it per data-protection obligations.

> **Priority:** export before any migration or deactivation — these entries are
> live business data and would otherwise be lost. See the
> [Customisation Audit](../../../01_Documentation/CUSTOMISATION_AUDIT.md).
