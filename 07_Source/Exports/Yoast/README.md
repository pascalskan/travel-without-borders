# Yoast SEO Export

## What belongs here

- Exported **Yoast SEO settings and per-page metadata** (titles, meta
  descriptions, Open Graph, schema) using Yoast's import/export tool.
- A short note recording the export date, Yoast version, and method.

## What does not belong here

- The Yoast plugin itself (vendor product).
- Sitemaps or generated output (regenerated by the plugin).

> Preserving this protects existing search rankings during any migration. Export
> before deactivation. See the
> [Customisation Audit](../../../01_Documentation/CUSTOMISATION_AUDIT.md).
