# Themes

Source for **custom and child themes** authored or owned by this project.

## What belongs here

- The **Ave Child** theme source (`functions.php`, `header.php`, `style.css`,
  template overrides, and any child-theme assets).
- Future custom themes or child themes we author.

## What does not belong here

- The **parent Ave theme** — it is a third-party product, restored from the
  vendor or a backup. Never edit or commit it.
- Any other vendor theme.
- Compiled/minified output that is generated from source elsewhere.

> The Ave Child `header.php` carries the hardcoded Google Tag Manager snippet —
> see [Tracking](../Tracking/README.md) and the
> [Customisation Audit](../../01_Documentation/CUSTOMISATION_AUDIT.md).
