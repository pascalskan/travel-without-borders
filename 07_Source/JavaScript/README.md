# JavaScript

Hand-written **custom scripts** that we author and maintain.

## What belongs here

- Custom JavaScript authored for the site, intended to be enqueued via the child
  theme with declared dependencies.

## What does not belong here

- Theme/plugin scripts shipped by vendors.
- Third-party tracking libraries loaded via a tag manager — see
  [Tracking/](../Tracking/README.md).
- Minified/compiled bundles (track the source, not the build output).

> The customisation audit found **no custom JavaScript** currently exists, so
> this is a clean, low-risk area for new work. See the
> [Coding Standards](../../01_Documentation/CODING_STANDARDS.md).
