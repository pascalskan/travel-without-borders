# Plugins

Source for **genuinely custom plugins** authored by this project.

## What belongs here

- Custom plugin source code we write (e.g. a small site-specific functionality
  plugin), each in its own subfolder.

## What does not belong here

- **Commercial or theme-bundled plugins** (Ave Core, Slider Revolution, WP
  Rocket, Quform, Yoast, etc.) — these are vendor products, restored from their
  source or a backup, never committed.
- Exported plugin *data* — that goes in [Exports/](../Exports/README.md).

> As of the customisation audit there are **no custom plugins** — all active
> plugins are commercial or theme-bundled. This folder is reserved for if that
> changes. See the
> [Customisation Audit](../../01_Documentation/CUSTOMISATION_AUDIT.md).
