# Roadmap

Planned phases for the Travel Without Borders redevelopment. Live progress is
tracked in the [Project Status](../PROJECT_STATUS.md) dashboard.

## Phase 0 — Discovery & Recovery ✅

Audit the site and establish a safe, recoverable baseline.

- Website, plugin, mobile, and theme audits.
- Baseline screenshots and verified full backup.
- Repository created.

## Phase 1 — Repository Housekeeping 🟡

Turn the repository into a professional, well-documented project.

- Standardise structure and naming.
- Complete core documentation and index.

## Phase 2 — Local Environment & Child Theme

Create a safe place to build.

- Stand up a local WordPress mirror.
- Replicate theme and plugins.
- Create the Ave child theme for all custom code.

## Phase 3 — Homepage Redesign

- Define goals and design direction.
- Implement via child theme / WPBakery.
- Validate against baseline screenshots.

## Phase 4 — Testimonials Redesign

- Define goals.
- Implement and test.

## Phase 5 — Optimisation

- Performance, mobile responsiveness, and accessibility.
- Re-test against baselines.

## Phase 6 — Deployment

- Pre-deployment backup.
- Follow the deployment checklist.
- Post-deployment verification and release notes.

## Guiding Principles

- **Backup first** — never change production without a current, verified backup.
- **Child theme only** — no edits to the parent theme or plugins.
- **Test against baselines** — every visual change is compared to captured
  baselines.
- **Document as you go** — decisions and changes are recorded.
