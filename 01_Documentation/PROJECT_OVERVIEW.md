# Project Overview

## Summary

Travel Without Borders is an existing WordPress website
([travelwithoutborders.co.uk](https://travelwithoutborders.co.uk)). This project
modernises and maintains the site through a controlled, backup-first,
documentation-driven workflow that protects the live environment at every step.

## Goals

- Redesign the homepage.
- Redesign the testimonials section.
- Improve performance, accessibility, and the mobile experience.
- Establish a maintainable child-theme workflow for custom code.
- Keep the production site recoverable at all times.

## Scope

**In scope**

- Custom CSS, JavaScript, and PHP delivered via an Ave child theme.
- Visual and content updates through WPBakery where appropriate.
- Documentation, testing evidence, and deployment process.

**Out of scope (for now)**

- Replacing the Ave theme or WPBakery.
- Migrating away from WordPress.
- Plugin replacement, unless required by an objective above.

## Context

The site uses the Ave theme with the WPBakery page builder on PHP 7.4.33. Full
audits (website, plugins, mobile, theme, and customisation) were completed in
Phase 0, along with a verified baseline backup. No development work has started
yet. Full platform detail is in [TECH_STACK.md](TECH_STACK.md); how the site is
customised is in [CUSTOMISATION_AUDIT.md](CUSTOMISATION_AUDIT.md).
